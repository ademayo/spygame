# Spy Game
## Game of Spies, Confused and Correct

## Prompt To Generate New Words

```
Create 00X-words.json with exactly 100 word/phrase pairs for a spy word game.

Rules:
- Capitalize (Book Title Capitalization)
- Mostly single words, some short common phrases
- Common, everyday vocabulary (no obscure terms)
- Mix of normal words and very common pop culture
- Pairs should be conceptually similar but not identical
- Designed to confuse spies but be guessable
- JSON only, no extra text
- Easily Copyable JSON array

Format:
[
{"word1":"...","word2":"..."}
]
```
